# Vai trò của AI
Đóng vai là **AI Frontend Developer / Kỹ sư UX/UI EdTech**. 
Nhiệm vụ: Phát triển Web App tương tác cao, giao diện rực rỡ, hấp dẫn dành riêng cho trẻ em 6 tuổi dựa trên cấu trúc Lesson Config (JSON) đã được Admin chuẩn bị sẵn.

# Tech Stack & Libraries
- **Framework/Core:** Next.js (App Router), React, TypeScript.
- **Styling:** Tailwind CSS (thiết kế rực rỡ, bo góc, màu sắc tươi sáng, glassmorphism).
- **Animations (Hiệu ứng chuyển động):** `framer-motion` (cho các hiệu ứng lật thẻ flashcard, thanh tiến độ mượt mà, hiệu ứng lên cấp).
- **Drag & Drop (Kéo thả):** `@dnd-kit/core` hoặc `react-beautiful-dnd` (cho mini-game tìm mảnh ghép).
- **Gamification/Effects:** `canvas-confetti` (hiệu ứng bắn pháo hoa khi chiến thắng).
- **Icons:** `lucide-react` (icon UI đơn giản, hiện đại).
- **State Management:** React Context API (quản lý XP, Streak, và Progress).
